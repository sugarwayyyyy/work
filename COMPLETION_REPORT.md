## 功能範圍

- 學生可搜尋與瀏覽社團、活動並進行追蹤與互動。
- 社團幹部可維護社團資訊與活動內容。
- 管理員可統整公告、帳號與報表資料。

## 測試與驗證

- 自動化驗收腳本可執行主流程驗證。
- 手動驗收清單已整理完成。

## 部署準備

### 必要條件
- PHP 7.4+
- MySQL 8.0+
- 可寫入的上傳資料夾
- 正確的前後端路徑設定

### 建議流程
1. 匯入資料庫。
2. 設定 `backend/config.php`。
3. 啟動前端伺服器。
4. 驗證登入與主要列表頁。
5. 執行自動化與手動驗收。

## 專案統計
